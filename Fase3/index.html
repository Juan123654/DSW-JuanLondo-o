Informacion General

La página contiene dos temáticas importantes para el desarrollo Web las cuales son: Desarrollo Wew Frameworks CSS y Posicionamiento Web, de manera sistemática y con múltiples adecuaciones se está realizando el montaje de la pagina web a medida que avanzamos en el curso Diseño de sitios web la página tendrá mejor visualización, experiencia e interactividad.

Proposito

El propósito es mostrar dos temáticas seleccionadas en la pagina web donde se realice la interacción del menú, muestre el contenido correcto y genere la mejor visualización posible para el usuario

Introduccion

En la siguiente página realizaremos la gestión de menú en el cual cada submenú nos dará su tema correspondiente mostrando de la mejor manera su visualización y contenido cargado correctamente para el aprendizaje de la Fase 3 del curso Diseño de sitios Web
